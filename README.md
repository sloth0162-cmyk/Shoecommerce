# 🖤 ShoeCommerce – Dark UI E-Commerce Website

🔗 Live Demo: https://sloth0162-cmyk.github.io/Shoecommerce/

## 📌 About The Project

ShoeCommerce is a **dark-themed e-commerce website UI** built using **HTML and CSS**.

This is my **first frontend project**, created to practice and demonstrate:

* UI design fundamentals
* Layout structuring
* Styling with CSS
* Building a clean and modern dark interface

The focus of this project is purely on **design and user interface**, not backend functionality.

---

## 🎯 Goals

* Build a real-world style e-commerce layout
* Improve my understanding of **HTML & CSS**
* Experiment with **dark UI design**
* Create a project that can evolve into a full-stack application

---

## ✨ Features

* Modern **dark theme UI**
* Product showcase section
* Clean navigation bar
* Styled buttons and hover effects
* Organized layout structure
* Basic responsive design
